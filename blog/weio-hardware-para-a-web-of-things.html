WeIO - O Hardware para a Web of Things

25/8/2014

WeIO is the new kid on the block

Um novo projeto promete revolucionar a maneira que desenvolvemos soluções para o mercado de Internet of Things. O WeIO é o mais novo Open Hardware do mercado.

IDE totalmente web

O principal diferencial do WeIO é a a interface de desinvolvimento (IDE) completamente desenvolvida em tecnologias web (HTML5, CSS3, JavaScript).

Com essa IDE o desenvolvedor consegue criar aplicações de maneira rápida e fácil utilizando jQuery e Bootstrap e o hardware ainda pode ser controlado pelo celular com poucas linhas de código.

Para desenvolver na IDE pode ser usado tanto Python como JavaScript, mas com certeza com o apoio da comunidade Open Source e Open Hardware o número de linguagens suportadas vai subir bastante.

O sistema operacional utilizado no projeto é o OpenWRT que é bem conhecido pela utilização para modificar roteadores. O servidor web utilizado é o Tornado, um servidor Python não bloqueante com capacidade de trabalhar com websockets e manter milhares de conexões abertas simultaneas.

Sobre o Hardware

Um das melhores coisas do WeIO é que ele já vem com um módulo de Wifi integrado.

As conexões físicas são:

  • 32 pinos GPIO (UART)
  • 2 pinos de interface serial síncrona (SPI)
  • 2 portas USB
  • 1 mini USB
  • 1 porta Ethernet
  • 1 termômetro (LM75)
  • 1 LED RGB

Open Source

O projeto pode ser encontrado no repositório aberto no GitHub.

Ele ainda não se encontra a venda mas logo vai ser uma ótima opção no mercado da "Web of Things".
